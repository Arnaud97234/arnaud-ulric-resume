## Getting Started

Start backend:

```bash
cd backend

yarn start
# or
nodemon
```

Run by default on [http://localhost:3000] (http://localhost:3000)

Start frontend:

```bash
cd frontend

yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser
If backend is already running on `:3000` switch to `:3001`:3001
