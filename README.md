## Getting Started

Start backend:

```bash
cd backend

yarn start
# or
nodemon
```

Run by default on [http://localhost:3000] (http://localhost:3000)

Start frontend:

```bash
cd frontend

yarn dev
```

Open [http://localhost:3001](http://localhost:3001) with your browser (If backend arlready running on `:3000`)
